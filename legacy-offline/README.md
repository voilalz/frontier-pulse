# 智域前沿离线使用说明

## 使用方法

1. 解压缩本项目。
2. 双击 `public/index.html`，使用 Chrome、Edge、Firefox 或 Safari 打开。
3. 页面浏览、搜索、筛选、排序、展开分析、收藏均不需要联网。

## 更新新闻数据

- 联网部署时，页面读取 `public/data/news.json`。
- 直接双击打开且远程数据不可用时，页面自动展示内置启动数据。
- 点击“导出数据”可以备份当前一期数据。

## 注意事项

- 离线打开不会自动抓取新闻；自动更新由 GitHub Actions 在线工作流完成。
- “阅读原文”链接只有在联网时才能打开。
- 收藏信息只保存在当前浏览器的本地存储中，不会上传。
- 新闻及原文版权归各来源所有，重要结论请进行多源交叉验证。
